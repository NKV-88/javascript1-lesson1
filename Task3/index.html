1. Чему будет равно JS-выражение 1000 + "108"?

Ответ: "1000108" (пример "answer = (1000 + '108');" если в консоли проверить переменную 'answer')

2. Самостоятельно разобраться с атрибутами тега script (async и defer)

Ответ:при использовании атрибута 'async' = браузер не останавливает обработку страницы, а спокойно работает дальше. Когда скрипт будет загружен – он выполнится. При использовании атрибута 'defer' = Первое – браузер гарантирует, что относительный порядок скриптов будет сохранён. Второе отличие – скрипт с defer сработает, когда весь HTML-документ будет обработан браузером.
